Hero Section
전면 사진 풀스크린 히어로
단순 텍스트+로고 배너에서 자연 사진 풀스크린으로 전환. Ken Burns 줌인 효과와 스크롤 패럴랙스로 첫인상 극대화.
BEFORE연두색 배경 + 로고 이미지 캐러셀
AFTERForest 사진 풀스크린 + 오버레이 + 패럴랙스
Color System
다크 그린 디자인 시스템
연두/베이지 CMS 배색에서 프리미엄 다크 그린(#061009 + #2a9a5a) 테마로 기관 신뢰도와 전문성 표현.
BEFORE#e8f0c0 연두 + #c8b884 베이지
AFTER#061009 딥다크 + #2a9a5a 에메랄드
Storytelling
문제→해결 스토리 구조
정보 나열형에서 문제(환경 위기 데이터) → 해결책(프로그램) → 신뢰(언론·인증) → CTA 흐름으로 설득력 있는 랜딩 구조.
BEFORE단순 기관 소개 + 기대효과 나열
AFTERProblem → Solution → Trust → CTA
Interaction
데이터 카드 + 탭 인터랙션
정적 텍스트에서 벤토 카드 3D 틸트, 카운터 애니메이션, 프로그램 탭 전환, 스크롤 reveal 등 풍부한 인터랙션 추가.
BEFORE정적 그리드 카드, 인터랙션 없음
AFTER벤토 3D 틸트 + 탭 + 카운터 + reveal
Trust Signal
신뢰 크레덴셜 전면 부각
예비사회적기업 지정, 양산일보 언론 보도, 경남 양산 지역 전문성을 별도 Trust 섹션으로 강조. 언론 기사 카드 직접 링크 포함.
BEFORE우측 상단 뱃지 텍스트만 표시
AFTER크레덴셜 3-card + 언론 기사 2건
Social Proof
인스타그램 실사진 3×3 그리드
@jaejae_lab 실제 활동 사진 9장을 그리드로 배치. 호버 시 인스타그램 오버레이, 클릭 시 계정 이동. 생동감 있는 활동 증거 제시.
BEFORE인스타그램 섹션 없음
AFTER실사진 3×3 그리드 + 팔로우 CTA
